# Install

1. git clone https://github.com/iam-in/khl.git
2. composer install
3. configure [database.global.php](app%2Fdatabase%2Fdatabase.global.php), [database.mig.php](app%2Fdatabase%2Fdatabase.mig.php)
4. vendor/bin/phpmig migrate


### 1. Run bubble sort
```bash
php ./app/index.php
```
